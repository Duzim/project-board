# Changelog

All notable changes to **My Project Dashboard** are documented here.
Todas as mudanças relevantes da extensão estão documentadas aqui.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Planned / Planejado
- Visual color picker inside the webview / Seletor de cor visual na webview.
- Drag-and-drop to reorder projects and groups / Arrastar para reordenar.
- Open-on-startup setting / Configuração para abrir na inicialização.

## [1.0.0] - 2026-06-30

### Added / Adicionado
- Initial release / Versão inicial.
- Project dashboard opened in the editor area / Painel de projetos na área do editor.
- Activity bar icon with a sidebar launcher / Ícone na barra de atividades com lançador na lateral.
- Groups to organize projects / Grupos para organizar projetos.
- Add, edit, and remove projects (name, path, color) / Adicionar, editar e remover projetos (nome, caminho, cor).
- Add, rename, and delete empty groups / Adicionar, renomear e excluir grupos vazios.
- Color selection via palette or custom hex/rgb / Cor via paleta ou hex/rgb personalizado.
- Click a card to open the project in a new window / Clique no card para abrir em nova janela.
- Persistent storage across sessions / Armazenamento persistente entre sessões.
